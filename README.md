### [🐊 Krokodil: AI Word Generator](https://github.com/ai-platon/krokodil)

> A party charades game powered by AI — available as a **web app** and a **native Android APK**

A browser party game — generate charades words for any topic using AI, play solo or with friends in real-time team mode. Started as a pure web project, then ported to **Android Studio** and packaged as a native `.apk`.

**Available on:**

| Platform | How to get it |
|---|---|
| 🌐 **Web** | [ai-platon.github.io/krokodil](https://ai-platon.github.io/krokodil/) — open in any browser |
| 📱 **Android APK** | Download `krokodil.apk` — install directly on any Android device |

What I learned building it:

- 🎨 Vanilla JS + CSS — multi-page SPA with animated dark UI, zero frameworks
- 🤖 Groq API + Llama 3.3 70b — AI word generation by topic, difficulty and count
- 🧠 Prompt engineering — tuning output format, difficulty levels and topic relevance
- 👥 Game logic — turn-based team mode with live countdown timer and round management
- 💾 JavaScript `localStorage` — per-topic leaderboard, generation history, API key
- 📱 Mobile-first — responsive bottom tab navigation, haptic feedback
- 🤖 **Android Studio** — WebView-based APK packaging, manifest configuration, APK build & signing

### [🛡️ AI Guard: Battle for Entry](https://github.com/ai-platon/ai-guard-game)

A browser jailbreak game — convince an AI guard to let you through using social engineering and clever prompts.

What I learned building it:

- 🎨 Vanilla JS + CSS animations — full cyberpunk UI from scratch, zero frameworks
- 🤖 Groq API — real-time AI responses with dynamic system prompts
- 🧠 Prompt engineering — designing AI characters with distinct personalities and vulnerabilities
- 💾 JavaScript `localStorage` — leaderboard and API key persistence
- 🌍 Multilingual AI — guards respond in the player's language automatically
- 🐙 GitHub Pages — deploying a static game with no backend
